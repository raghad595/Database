# 🗄️ MySQL Database Solutions

A collection of database problems, schema designs, and querying solutions implemented in MySQL. This repository demonstrates relational database modeling, entity-relationship diagram (ERD) translation, and data management across different domains.

## 📁 Repository Structure

*   **`airline_erd.sql`**: Contains the SQL scripts for an Airline management system. This file defines the architecture and relationships for handling entities such as flights, passengers, aircraft, and bookings.
*   **`grade_management.sql`**: Contains the schema and queries for an academic grading system. This file handles the relational logic for managing student profiles, course enrollments, and academic grade records.

## 🚀 Getting Started

To run or interact with these databases on your local machine:

1.  Ensure you have a MySQL server installed and running (e.g., via MySQL Workbench, XAMPP, or the MySQL CLI).
2.  Open your preferred SQL client and connect to your local server.
3.  Import or execute the scripts to generate the respective schemas and tables:
    ```sql
    SOURCE /path/to/airline_erd.sql;
    SOURCE /path/to/grade_management.sql;
    ```

## 🛠️ Tech Stack

*   **Database Management System:** MySQL
*   **Language:** SQL (DDL, DML, and DQL)
